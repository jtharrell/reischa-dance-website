Beginner / Intermediate Bellydance:

Beginner / Intermediate Level

This is an introduction to the fundamentals of belly dance used in all styles. Fundamentals will be covered through a series for basic technique exercises, combinations, and choreography. This is a beginner level class, but it is also a wonderful refresher course. This class will include some Intermediate level information.

Intermediate / Advanced Bellydance:
This class is not meeting until further notice. For the time being, the Intermediate Class is being integrated into the Beginner Class listed above.

This is a continuation of belly dance fundamentals. Challenging exercises, combinations, choreography, and even improvisation will be studied. This is an intermediate level class recommended to students with prior study.
